# Features

## Core experiences
- AI persona introduction: a conversational landing point instead of a static hero section
- Guided recruiter onboarding: step-by-step prompts, use cases, and persona context
- Interactive project storytelling: AI summaries with context-aware detail
- Skill discovery via dialogue: ability to ask about technologies, achievements, and values

## AI capabilities
- OpenAI-powered chat with persona-specific prompts
- Context retention across conversation sessions
- Future voice synthesis using ElevenLabs
- Query handling for technical, career, and project questions

## Data and integrations
- Supabase for developer data persistence, auth, and user context
- OpenAI for generative responses and conversational guidance
- ElevenLabs for text-to-speech output and persona voice
- Future Qdrant for semantic search, embeddings, and RAG
- Future FastAPI backend for content orchestration and secure AI proxies

## Quality features
- Responsive layout and accessibility-first interactions
- Scalable design system with reusable UI primitives
- Performance-aware animation and rendering strategy
- Developer-friendly architecture and typed data contracts
